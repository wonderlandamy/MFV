# MFV
This Repo. is used for our The Visual Computer paper:

Enhanced Person Re-Identification through Multi-Scale Feature Fusion and Attention Mechanisms

For using the codes, please download the public Market-1501、 Duke、 MSMT17、 CUHK03 datasets. Besides, please install the Pytorch as the Official Suggestions.

Then, run the train.py and test.py with your GPUs. You can get the same results in the above paper.
